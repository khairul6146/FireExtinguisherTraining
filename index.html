3. Operation Method (PASS)

Bagaimana cara menggunakannya?

PULL

Tarik pin keselamatan

Break the seal and pull the pin. / Pecahkan kunci dan tarik pin.

AIM

Hala muncung ke pangkal api

Aim at the base of the fire. / Tala ke pangkal api.

SQUEEZE

Tekan pemegang

Squeeze the lever slowly. / Tekan pemegang perlahan-lahan.

SWEEP

Sapu dari sisi ke sisi

Move side to side. / Sapu dari kiri ke kanan.
