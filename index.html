3. Fire Blanket

Selimut Api - Smothering Method

Safety Protocol / Protokol Keselamatan

👐

1. PULL

Pull tapes down to release.

Tarik pita ke bawah.

🛡️

2. SHIELD

Protect hands with edges.

Lindungi tangan anda.

🥘

3. COVER

Place gently over fire.

Tutup api perlahan.

⏱️

4. WAIT

Leave until cool.

Biar sehingga sejuk.

🍳Class K / Kitchen
🏃Person in distress

4. Correct Technique (PASS)

Tatacara Penggunaan Pemadam Api yang Betul

PULL

Tarik Pin

Break the seal and pull the safety pin out completely.

AIM

Hala Pangkal Api

Point the nozzle at the base of the fire, not the flames.

SQUEEZE

Tekan Tuas

Press the lever to discharge. Release to stop spray.

SWEEP

Sapu Kiri-Kanan

Sweep the nozzle side to side across the base of the fire.
